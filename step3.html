Before starting this step, check out if things are working. Now that Abacus is up and running, open the homepage. Create a new user, add stuffs, check logins, delete permissions for file If you're getting errors, try troubleshooting, or can email me at prasilkoirala@hotmail.com and explain the error you are getting. If everything's working good, let's start on customization.

The project is split into modules where each module contribute to a certain part of the website. Customization involves editing and adding information basically to these files -
  • top.php - Persistent top bar
  • index.php - The home page
  • myinfo.php - About page
  • help.php - Help page
  • team.php - Team page
  • footer.php - Website footer
  • abacus_style.css - CSS stylesheet
That all. You've successfully set up Abacus. Goto Home
