Before beginning, make sure the PHP on your server is configured properly. You can use this guide if you need help. If you bought a dedicated or shared server, it is likely to have a PHP parser installed already.

Abacus, by default uses MySQL for storing data. So you need to set up MySQL for your server. Refer this page if you need help. Again, if you bought a dedicated or shared server, MySQL is likely to be installed already.

Download Source Files
  1. Upload the all files and folders in the "Source" folder into the main folder.
  2. Create a database named "abacus".
  3. Import "abacus_tables.sql" to abacus database.
